# Video Subtitle Generator

### Aplikacja do automatycznego generowania napisów z plików wideo z wykorzystaniem modeli speech-to-text.

#### Projekt umożliwia użytkownikowi wgranie pliku wideo, automatyczne wyodrębnienie ścieżki audio, transkrypcję mowy na tekst oraz wyświetlenie wideo zsynchronizowanego z wygenerowanymi napisami. Celem aplikacji jest szybkie i dostępne tworzenie napisów bez potrzeby ręcznej edycji.
